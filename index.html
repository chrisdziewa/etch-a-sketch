Controls:

Use the mouse to hover over the squares to change their lightness values. Stop and start pen flow by hitting the SPACEBAR

This project was made from scratch using HTML5, CSS3, JS, and jQuery.

Modes:

Normal Mode uses a black field and each stroke becomes lighter

Random Color Mode changes each block to a randomly generated color
